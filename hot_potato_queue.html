Hot Potato — Simulasi Queue

Permainan Hot Potato diimplementasikan menggunakan Queue FIFO — operasi enqueue() dan dequeue() mensimulasikan oper kentang melingkar hingga tersisa satu pemenang.

Cara Kerja Queue
enqueue() + dequeue()
Oper kentang: dequeue dari front, enqueue ke rear — mensimulasikan lingkaran.
dequeue() tanpa enqueue()
Setelah N oper, pemain di front langsung disingkir dan tidak dikembalikan.
queue.size() == 1
Kondisi berhenti — sisa satu pemain adalah pemenang.
Implementasi Python
hot_potato.py 
def hot_potato(names, num):
    # Masukkan semua pemain ke queue
    q = Queue()
    for name in names:
        q.enqueue(name)          # enqueue() tiap pemain

    while len(q) > 1:
        # Oper kentang: dequeue depan → enqueue belakang
        for _ in range(num):
            q.enqueue(q.dequeue())   # simulasi oper melingkar

        q.dequeue()              # tersingkir! (tanpa enqueue)

    return q.dequeue()          # satu pemain tersisa = pemenang


# Contoh penggunaan
pemain = ["Budi", "Ani", "Citra", "Dedi", "Eka", "Fani"]
pemenang = hot_potato(pemain, 4)
print(f"Pemenang: {pemenang}")
